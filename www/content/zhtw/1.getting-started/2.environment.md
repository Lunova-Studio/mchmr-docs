#title
闲言碎语

#content
  ::div{class="mb-2"}
  McHMR 管理端不建议部署在你的游戏服务器主机上，当玩家在更新时，会大量占用你的主机资源，如果你的主机性能不高，可能会影响游戏服务器的 TPS 和玩家的游戏体验。
  ::

  ::div{class="mb-2"}
  我们建议将 McHMR 部署在非游戏服务器的主机上。
  ::

  ::div
  我们推荐部署到常见的 Linux 发行版系统，如：Ubuntu、CentOS、Debian 等，部署 McHMR，可以尽可能发挥 McHMR 的性能。或者使用 Windows Server 系统。
  ::
::


## 安装依赖环境

::tabs
  ::div{label="Windows Server" icon="logos:microsoft-windows-icon"}
  为了安装更加方便，在 Windows Server 上，你可以安装[phpStudy](https://www.xp.cn/phpstudy)，它包含了 McHMR 所需的 MySQL 和 Nginx 的依赖环境（Java 和 Redis 你仍然需要手动安装），你可以直接在 phpStudy 上安装这些依赖环境，如果你不想使用 phpStudy，也可以参考以下提供的安装方法。

  我们在博客园中找到了几篇详细安装 Java 1.8、MySQL 8.0 和 Nginx 的文章，你可以参考这些文章进行安装。
  :read-more{icon="vscode-icons:file-type-java" title="Java 1.8 安装教程" target="_blank" to="https://www.cnblogs.com/nojacky/p/9497724.html"}
  :read-more{icon="logos:mysql-icon" title="MySQL 8.0 安装教程" target="_blank" to="https://www.cnblogs.com/haha029/p/16855562.html"}
  :read-more{icon="vscode-icons:file-type-nginx" title="Nginx 安装教程" target="_blank" to="https://www.cnblogs.com/taiyonghai/p/9402734.html"}
  我们在 GitHub 的 Redis 项目中找到了最后支持 Windows 操作系统的 Redis 版本，你可以点击下方卡片，在 `Assets` 中，选择 `Redis-x64-5.0.14.1.msi` 下载。
  :read-more{icon="logos:redis" title="Redis Github 仓库" target="_blank" to="https://github.com/tporadowski/redis/releases"}
  ::
  ::div{label="Linux" icon="logos:linux-tux"}
  为了安装更加方便，在 Linux 上，你可以通过安装[宝塔面板](https://www.bt.cn/pc/home)、[1Panel](https://www.1panel.cn/) 等面板来安装 McHMR 所需的依赖环境（Java、MySQL、Nginx、Redis），你可以直接在面板上安装这些依赖环境，如果你不想使用面板，也可以参考以下提供的安装方法。

  我们在博客园和腾讯云开发者社区中找到了几篇详细安装 Java 1.8、MySQL 8.0、Nginx 和 Redis 的文章，你可以参考这些文章进行安装。
  :read-more{icon="vscode-icons:file-type-java" title="Java 1.8 安装教程" target="_blank" to="https://www.cnblogs.com/cao-lei/p/14768643.html"}
  :read-more{icon="logos:mysql-icon" title="MySQL 8.0 安装教程" target="_blank" to="https://www.cnblogs.com/MrYoodb/p/15811199.html"}
  :read-more{icon="vscode-icons:file-type-nginx" title="Nginx 安装教程" target="_blank" to="https://www.cnblogs.com/orangebooks/p/12058830.html"}
  :read-more{icon="logos:redis" title="Redis 安装教程" target="_blank" to="https://cloud.tencent.com/developer/article/2291807"}
  ::
::
