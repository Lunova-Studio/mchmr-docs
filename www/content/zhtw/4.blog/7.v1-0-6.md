## 新内容
1. 优化创建版本逻辑，大幅提高创建速度
2. 修复文件名为中文时可能导致下载失败的问题
3. 允许用户自主选择是否启用客户端登录限制
4. 白名单支持通配符*(测试中，可能不稳定)
5. 移除白名单不能为空的限制

### 更新说明
::alert{type="info" icon="lucide:info"}
  1.0.6 公测包中已携带 SQL 更新文件（Fix 1.0.5.sql）
::

下载最新版本的 McHMR 后，打开`管理端后端`文件夹，进入`SQL`文件夹，在终端中运行SQL执行命令，更新 mchmr 数据库。
```bash
mysql -h localhost -u {数据库用户名} -p -D mchmr < "Fix 1.0.5.sql"
```
或者使用 MySQL 客户端工具，如 Navicat 等，执行 SQL 文件进行更新。

其他文件直接替换即可。

::alert{type="warning" icon="lucide:triangle-alert"}
  1. 请备份数据库后再进行更新；
  3. 若修改了启动脚本，需要根据新脚本重新配置启动参数。
::
