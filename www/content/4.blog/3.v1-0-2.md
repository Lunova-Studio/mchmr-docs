## 新内容
1. 优化重复 IP 获取 Token 的提示
2. 优化选择启动器时自动打开的文件夹为当前程序运行目录
3. 修复更新器自动切换深色模式异常
4. 修复 .minecraft 文件夹拼写错误
5. 修复无法生成 API 的问题

### 更新说明
::alert{type="info" icon="lucide:info"}
  1.0.2 公测包中已携带 SQL 更新文件（Fix 1.0.1.sql）
::

下载最新版本的 McHMR 后，打开`管理端后端`文件夹，进入`SQL`文件夹，在终端中运行SQL执行命令，更新 mchmr 数据库。
```bash
mysql -h localhost -u {数据库用户名} -p -D mchmr < "Fix 1.0.1.sql"
```
或者使用 MySQL 客户端工具，如 Navicat 等，执行 SQL 文件进行更新。

其他文件直接替换即可。

::alert{type="warning" icon="lucide:triangle-alert"}
  1. 请备份数据库后再进行更新；
  3. 若修改了启动脚本，需要根据新脚本重新配置启动参数。
::
